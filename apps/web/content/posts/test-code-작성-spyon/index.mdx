- test code 작성 spyOn()

  ```tsx
  // chargeCash가 호출되고 있는지 감시하는 로직
  spyOn(GTagManager, 'chargeCash');
  
  // check result
  expect(GTagManager.chargeCash).toHaveBeenCalled();
  ```
